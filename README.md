## Python for Data Science
A structured mini-course covering core Python skills and data workflows used in data science.

### 00 — Starting
Introductory material on Python basics, setup, and essential syntax for data tasks.

### 01 — Array
Working with arrays and their manipulations, including common operations used in numerical computing and image handling.

### 02 — DataTable
Loading, cleaning, manipulating, and displaying tabular datasets to prepare data for analysis.

### 03 — OOP
Object-oriented design in Python: classes, inheritance, and structuring code for reusable data pipelines.

### 04 — DoD (Design of Data)
Data structure design and modeling best practices for robust data processing and analysis workflows.

## darkly — Web Security
A hands-on web security project that involves dissecting a deliberately vulnerable website to identify common flaws.
Focuses on developing a security-minded approach to web apps, covering exploitation techniques, root causes in code/design, and defensive thinking.
